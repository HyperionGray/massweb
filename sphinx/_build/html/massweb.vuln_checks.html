match¶

Helper functions for matching parts of strings

massweb.vuln_checks.match.match_string(str_in, match)¶

Check if a string is found in another string.

str_in Input string. match String to find in str_in.

returns True if a match is found and False if no match is found.

massweb.vuln_checks.match.match_strings(str_in, match_list)¶

Check if any of a list of strings is found in another string.

str_in input string. match_list list of strings to find in str_in.

returns True if a match is found and False if no match is found.

massweb.vuln_checks.match.parse_match(str_in, tag, match)¶
Parse HTML string for a particular tag and attempt to find a matching
string in the contents of that tag.

Currently is hardcoded for ‘script’ tags.

str_in input string. tag html tag to parse for. match string to match against the contents of the tags.

returns True if a match is found and False if no match is found.

Installation¶

PIP (Recommended)¶

To install MassWeb with pip, simply type pip install massweb and you should get a working install on your system.

Manual¶

To install MassWeb from source: #. Install MassWeb’s dependencies.

  • requests
  • beautifulsoup4
  • html5lib

  1. Download MassWeb here.

  2. Unpack the archive.

  3. Install:

    cd massweb
python setup.py install

Development¶

For development it is recommended that you use virtualenv. Below are the steps to start building and testing.

  1. Clone the git repo.

  2. Add virtualenv directory to your copy of the repo:

    $ cd massweb
$ virtualenv env

  3. Activate the virtualenv and run the refresh.sh script: * If you are working on the documentation export REFRESH_SPHINX=true before running refresh.sh.

    $ source env/bin/activate
$ ./test/refresh.sh

  4. Make changes.

  5. Run refresh.sh again to build and install in a clean virtualenv.

  6. Test changes.

url_generator¶

Some helpers to generate URLs for fuzzing.

massweb.fuzz_generators.url_generator.append_to_param(url, param, value)¶

Replace a parameter in a url with another string.

Return a fully reassembled url as a string.

url URL to mangle as string. param Parameter in url to replace the value of. value String to replace the value of param in url with.

massweb.fuzz_generators.url_generator.geneerate_fuzzy_urls(url, payloads)¶

Generate a URL suitable for fuzzing, using the suplied payloads.

url Base URL. payloads list of Payload objects to apply to url. returns list of URLs as strings
